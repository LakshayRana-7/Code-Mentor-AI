# CodeMentor AI

## Overview
AI-powered code review platform built with the MERN stack.

## Features
- Authentication
- Monaco Editor
- AI Review API
- Review History
- User Profile
- Dark Mode
- PDF Export

## Run

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev

## Deployment
Frontend: Vercel
Backend: Render
Database: MongoDB Atlas
